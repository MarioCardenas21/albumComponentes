Álbum de Componentes de PC (Ilustrativo)
======================================

Estructura del proyecto:
- index.html
- styles.css
- app.js
- img/  (coloca aquí tus imágenes; este ZIP trae un marcador para que la carpeta no quede vacía)

Cómo usar:
1) Descomprime el ZIP en una carpeta.
2) Abre index.html en tu navegador.
3) Agrega imágenes dentro de img/ y ajusta las rutas en app.js (propiedad `image` de cada objeto del arreglo DEMO).
4) Puedes crear subcarpetas dentro de img/ (cpu, gpu, ram, etc.). El código tiene un *fallback* (placeholder) si una imagen no existe.

Notas:
- Todo es ilustrativo, sin precios.
- Orden por nombre (A→Z/Z→A), búsqueda y filtro por categoría.
